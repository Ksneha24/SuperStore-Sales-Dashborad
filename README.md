# SuperStore-Sales-Dashborad
🚀 Project Overview
This project presents a dynamic and interactive Sales Performance Dashboard built using Power BI to analyze and visualize key business metrics from a Superstore dataset.
The dashboard helps stakeholders track sales, profit, and customer trends, enabling data-driven decision-making.

🎯 Objectives
Monitor overall sales and profit performance
Identify top-performing products and regions
Analyze customer segments and purchasing behavior
Track monthly and yearly growth trends
Detect loss-making categories and improvement areas

📁 Dataset Description
The dataset includes retail sales data with the following key columns:
Order ID
Order Date
Ship Mode
Customer Name
Segment (Consumer, Corporate, Home Office)
Region (East, West, Central, South)
Category & Sub-Category
Sales
Quantity
Discount
Profit

⚙️ Tools & Technologies Used
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions) (KPI Calculations)
Excel/CSV dataset

Key Insights
Identified high revenue regions and top-selling categories
Found loss-making sub-categories due to high discounts
Consumer segment contributes the highest sales volume
Seasonal trends observed in monthly sales patterns

💡 Business Impact
Helps management optimize pricing and discount strategies
Enables better inventory and sales planning
Improves customer targeting and segmentation
Supports real-time performance monitoring
